Let's go over some definitions. Click on the terms below to see a description.

Javascript

JavaScript is a programming language used to make web pages interactive.

Operators

An operator is a special character (or characters) that indicates an action to be performed. +,-,*,/, and % are all operators.

Variables

A variable is sort of like a container for storing values. For example in var myNumber = 3; myNumber is a variable that we set equal to 3.

Variable naming conventions

In JavaScript, it is conventional to give variables names that start with a lowercase letter, and if the name has more than one word, to capitalize each word after the first, without spaces or punctuation (such as hyphens - or underscores _) between words. This capitalization style is called lower camelcase, named after the humps of a camel. Example: lowerCamelCase

Functions

Like a method, a function is something that performs an action. Functions are different from methods because they are called by themselves, whereas methods are actions that are called on something.

Strings

A series of characters inside of quotation marks, like "Hello world", is called a string. Strings can contain any number of letters, punctuation, and even numbers. They represent text

Methods

A method is an action that is taken on something. For example console.log() takes an action on the JavaScript console: it logs whatever we put inside of the parenthesis to the console.

Arguments

The part of the method inside of the parenthesis is called an argument. Arguments give methods more information about what they are supposed to do. In the case of console.log, the argument is what we want to print to the console.
